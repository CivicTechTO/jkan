About this portal

This is a community-maintained effort to collect all Toronto datasets in a single system, whether they originate from the city or from citizens.

The city-maintained datasets are collected through the use of automated web scrapers that the community maintains together.

Goals

  • Collect official datasets in one place.Different arms of the city run various data portals. These can be merged into one place.
  • Allow citizen datasets as first-class objects.Citizens often create their own datasets. These should live in the same system as the city's officially released datasets.
  • Allow citizens to share transformed datasets. Citizens often write scripts to transform data to clean it up or transform it into more useful file formats. These should be available along-side the original resources.
  • Normalize dataset and organizational metadata. Lots of metadata on the portal seems to be entered freeform, and so there are inconsistencies that we can resolve.
  • Implement a dataset versioning system. Many official datasets will disappear, or update without notice. The community would benefit from a caching layer to provide change management.
